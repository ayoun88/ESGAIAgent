# 🛠️ Tools

`agent.py`의 ReAct 에이전트가 호출하는 Custom Tool 모듈 모음입니다. 총 16개 도구들에 대한 개요는 아래와 같습니다.
---

## 📚 RAG / 검색 도구

### `search_pdf.py` — 고아연
PDF 가이드라인·보고서에서 근거 문서를 찾는 RAG 검색 도구입니다. KoNLPy 형태소 분석 기반 BM25와 ChromaDB 시맨틱 검색을 0.5:0.5로 앙상블하고, 메타데이터 필터(연도/기업/문서 카테고리)를 4단계로 점진 완화하는 폴백 검색을 적용했습니다.

### `search_csv.py` — 고아연
배출계수 등 정형 데이터를 SQL로 조회하는 Text-to-SQL 도구입니다. LangGraph 3단계 파이프라인(`generate_query → execute_query → should_retry`)으로 구성되어, 쿼리 실행 중 에러가 발생하면 최대 2회까지 자가 복구 재시도를 수행합니다.

### `web_search.py` — 김재현
Tavily API를 활용한 범용 실시간 웹 검색 도구입니다. 실시간 환율, ESG 규제 동향, 최신 뉴스 등 RAG로 커버할 수 없는 최신 정보를 조회할 때 호출됩니다.

### `search_esg_regulation.py` — 윤소정
ESG 법령·규제에 특화된 검색 도구입니다. 입력 파라미터(예: 이사회 독립성, 공시 의무)로 검색 쿼리를 구성해 웹 검색을 호출하고, 공식 출처(금융위원회, 고용노동부, 국가법령정보센터, 전자공시시스템)와 미공식 출처를 구분해 신뢰도를 태깅합니다.

## 🌍 환경(E) 계산 도구

### `calculate_carbon.py` — 이진성
Scope 1/2/3 탄소 배출량 계산기입니다. 초기에는 하드코딩된 값으로 로직만 검증했으나, 이후 SQL DB(국내 LCI·DEFRA·EPA 배출계수)를 직접 조회하도록 변경되어 배출계수가 바뀌어도 코드 수정 없이 데이터만 갈아끼우면 되도록 설계했습니다.

### `water_recycling.py` — 배준서
용수 재활용률과 자원 순환율(폐기물 재활용)을 계산하는 도구입니다. GRI 303/306 기준으로 산업별 벤치마킹 데이터와 비교한 진단 리포트를 마크다운으로 반환합니다.

---

## 👥 사회(S) 계산 도구

### `calculate_safety.py` — 박성재
산업안전 지표인 LTIR(휴업재해율)·TRIR(전체재해율)을 계산합니다. OSHA 표준 단위(200,000 man-hours)를 사용하며 중대재해처벌법 / K-ESG S-4-2를 근거로 합니다.

### `calculate_employee_kpi.py` — 윤소정, 박성재
임직원 다양성 KPI 4종(성별·연령대·장애인 고용률·고용형태 비율)을 계산하고 K-ESG 1~5단계 점수까지 산출합니다. 박성재님이 기업 규모(소/중/중견/대기업) 판정과 규모별 법정 의무교육 체크리스트, 장애인 의무고용 개선 권고 로직(`labor_law`)을 추가했습니다.

## 🏛️ 지배구조(G) 계산/검색 도구

### `score_ethics_risk.py` — 윤소정
K-ESG G-4-1의 5개 카테고리(윤리규범 수립, 교육·서명, 신고 채널, 신고자 보호, 위반 처리)를 체크리스트화해 위험 등급(LOW~CRITICAL)을 산출합니다.

### `governance_benchmark.py` — 배준서
DART Open API와 연동해 동종업계 리딩 기업의 지배구조 지표(이사회 구성, 위원회 현황 등)를 벤치마킹합니다. API 승인 대기 등으로 호출이 어려울 때는 Mock 데이터로 자동 전환됩니다.

### `analyze_security_compliance_gap.py` — 이진성
정보 보호 인증 및 사고 대응 체크 도구입니다. 가이드가 필요한 질문은 RAG로 PDF를 검색해 안내하고, 체크 항목이 필요한 질문은 전처리된 CSV(ISO 27001, ISMS-P 체크리스트)를 조회해 누락된 항목을 마크다운으로 안내합니다.

## 🧩 통합 / 공용 도구

### `compliance_gap_analysis.py` — 윤소정
위 계산 도구들의 결과값(K-ESG 단계, 리스크 등급 등)을 재활용해, 기업의 ESG 충족 현황과 개선 우선순위(법적 의무 → K-ESG 단계 순)를 종합 리포트로 출력합니다.

### `csv_loader.py` — 김재현
사용자가 업로드한 CSV를 인코딩 자동 감지(UTF-8 → EUC-KR) 후 읽고, 회사마다 다른 컬럼명을 표준 컬럼명으로 매핑해 각 계산 도구가 그대로 사용할 수 있는 형태로 변환합니다.

### `report_generator.py` — 김재현
에이전트의 마크다운 답변을 Word(.docx) 문서로 변환해 저장합니다. 자동 반복 호출 문제를 막기 위해 사용자가 명시적으로 저장을 요청할 때만 동작하도록 독스트링에 가드레일을 명시했습니다.

### `calculate_demo.py` — 김재현
여러 항목을 한 번에 입력받아 처리할 수 있게 만든 배치 계산 도구로, 툴 호출 횟수와 토큰 사용량을 줄이기 위한 용도입니다.

### `__init__.py` — 공통
위 Tool 모듈들을 에이전트가 호출할 수 있도록 등록합니다.
